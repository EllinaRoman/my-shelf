# 📖 My Shelf — Твой уютный книжный уголок

**My Shelf** — это веб-приложение для книголюбов, которое помогает структурировать личную библиотеку, отслеживать прогресс чтения и сохранять любимые тропы.

## ✨ Особенности
- **Две темы:** Нежная светлая и атмосферная темная.
- **Умная форма:** Добавление книг с указанием жанров, тропов и личных заметок.
- **Статусы чтения:** "Читаю", "Прочитано", "Хочу прочитать" и другие.
- **Личный архив:** Удобное отображение карточек книг с рейтингом.

## 🎨 Стек технологий
- **HTML5** (Семантическая верстка)
- **CSS3** (Custom Properties для тем, Flexbox/Grid)
- **JavaScript** (Логика модальных окон и фильтрации)

## 🚀 Как запустить
1. Склонируйте репозиторий: 
   `git clone https://github.com/EllinaRoman/my-shelf.git`
2. Откройте `index.html` в вашем браузере.
